# Kiberxavfsizlik Alifbosi WebAR demo

1) `assets/poster.png` faylini MindAR Image Target Compiler orqali `.mind` formatga aylantiring:
https://hiukim.github.io/mind-ar-js-doc/tools/compile/

2) Chiqqan faylni `assets/targets.mind` nomi bilan shu papkaga qo'ying.

3) Local test:
```bash
cd webar_kiber_alifbo_demo
python -m http.server 8000
```
Kompyuterda: http://localhost:8000

4) Telefonda ishlashi uchun HTTPS hosting kerak:
GitHub Pages, Netlify yoki Vercel.

Demo hozir F, N, P, Q, X hotspotlari bilan ishlaydi. Keyingi bosqichda barcha harflar koordinatalarini qo'shamiz.
